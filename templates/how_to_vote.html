{% blocktrans %}About Voting{% endblocktrans %}

{% blocktrans %}Cambridge has a ranked choice system, which might be different than what you’re used to. Essentially, you'll be given a ballot with a list of the candidates, and you will fill in a bubble for the rank you want to give them.{% endblocktrans %}

So you'll start with something like this:

Seabiscuit 1 2 3
Phar Lap 1 2 3
Secretariat 1 2 3

And turn in something like this:

Seabiscuit 1 2 3
Phar Lap 2 3
Secretariat 1 3
